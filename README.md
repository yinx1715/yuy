# 📚 背單字軟體 - Word Learning App

一個純前端的英文單字學習應用，使用 HTML、CSS 和 JavaScript 開發。

## 🎯 功能特性

### 1. 主學習頁面 (`index.html`)
- **互動式卡片翻轉**: 點擊卡片實現 3D 翻轉效果
- **卡片正面**: 顯示英文單字
- **卡片背面**: 顯示中文解釋、詞性、字根分析等詳細信息
- **導航控制**: 
  - 點擊「上一個」「下一個」按鈕瀏覽單字
  - 進度計數器顯示當前位置（如 3/10）
- **快捷鍵支持**:
  - `Space` - 翻轉卡片
  - `←` / `→` - 上一個/下一個單字

### 2. 單字管理頁面 (`manage.html`)
- **新增單字表單**:
  - 英文單字（必填）
  - 中文解釋（必填）
  - 詞性（可選：名詞、動詞、形容詞等）
  - 字根分析（可選）
  - 例句（可選）

- **✨ 自動填入功能**:
  - 點擊「自動填入」按鈕
  - 自動呼叫多個 API：
    - **Dictionary API**: 獲取英文單字的詞性和例句
    - **翻譯 API**: 自動生成中文翻譯
    - **字根分析**: 智能分析常見的英文詞根和後綴
  - 自動填入的字段可以手動修改

- **單字列表**:
  - 顯示所有已添加的單字
  - 支援編輯和刪除功能
  - 顯示添加時間和詞性信息

## 🚀 快速開始

### 方式一：直接開啟文件
1. 將所有文件保存在同一個文件夾
2. 用瀏覽器打開 `index.html`

### 方式二：使用本地伺服器（推薦）
```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js 的 http-server
npx http-server
```
然後在瀏覽器中打開 `http://localhost:8000`

## 💾 數據存儲

- 所有單字數據存儲在瀏覽器的 **LocalStorage** 中
- 數據不會上傳到服務器
- 清除瀏覽器數據將刪除所有單字
- 支持跨頁面數據同步

## 🔧 自動填入 API

應用使用以下免費 API 進行自動填入：

1. **Free Dictionary API**
   - 來源: https://api.dictionaryapi.dev
   - 用途: 獲取英文單字的詞性、定義和例句

2. **MyMemory Translation API**
   - 來源: https://mymemory.translated.net
   - 用途: 將英文翻譯成繁體中文

3. **本地字根分析**
   - 預設了常見的英文詞根和後綴
   - 自動分析和解釋

## 🐛 故障排除

### 自動填入不工作
- 檢查網絡連接
- 確保輸入的是英文單字
- 等待 API 響應（可能需要幾秒鐘）
- 如果 API 不可用，手動填寫信息

### 單字未保存
- 檢查瀏覽器是否允許 LocalStorage
- 確保使用 http:// 或 https:// 協議（不是 file://）
- 嘗試使用本地伺服器方式打開

### 卡片翻轉效果不正常
- 升級瀏覽器到最新版本
- 清除瀏覽器緩存
- 嘗試其他瀏覽器測試
